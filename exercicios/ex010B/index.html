Link em outra janela

Drivers license - Olivia Rodrigo

No link acima foi usado target="_blank" para especificar que deveria ser feito em outra janela

Há também as opções _self (na mesma página, mas é o padrão caso nada seja especificado), _top, _parent. As duas últimas envolvem o uso de frames que segundo o Guanabara está caindo em desuso.

Hanging on the telephone - Blondie

No link acima foi usado rel="external" para especificar que o link é para um site externo.

Próxima página do site

Foi usado rel="next" para especificar que o link é para próxima página do próprio site.

Há também as possibilidade de usar como valores para o parâmetro rel="" os seguintes: prev(usado na página 2 no link de volta a esta página), author(link para o site do autor do artigo atual)
